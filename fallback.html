Réveil manuel

Si le bouton d'allumage ne répond pas dans l'app, vous pouvez réveiller le serveur depuis votre téléphone ou ordinateur avec une de ces apps gratuites.

Vos paramètres

Cliquez sur une valeur pour la copier, puis collez-la dans l'app que vous choisissez ci-dessous.
— Copie auto indisponible — sélectionnez le texte à la main
— Copie auto indisponible — sélectionnez le texte à la main
9 Copie auto indisponible — sélectionnez le texte à la main

📱 Android

WolOn — gratuit, Play Store
  1. Installez WolOn depuis le Play Store
  2. Ouvrez l'app, ajoutez un nouvel appareil
  3. Collez la MAC, le domaine et le port (en haut de cette page)
  4. Sauvegardez, puis cliquez sur l'appareil pour le réveiller
Chercher WolOn

🍎 iPhone / iPad

Mocha WOL — gratuit, App Store
  1. Installez Mocha WOL depuis l'App Store
  2. Ajoutez un appareil
  3. Collez la MAC, le domaine (champ Host ou IP) et le port
  4. Sauvegardez, puis cliquez sur l'appareil pour le réveiller
Chercher Mocha WOL

💻 Windows

PowerShell — intégré à Windows, aucune installation
  1. Ouvrez le menu Démarrer, tapez PowerShell et lancez-le
  2. Copiez la commande ci-dessous (déjà remplie avec vos paramètres)
  3. Collez-la dans PowerShell et appuyez sur Entrée
$mac=[byte[]]-split('AABBCCDDEEFF' -replace '..','0x$0 ');$u=New-Object Net.Sockets.UdpClient;$u.Connect('myserver.example.com',9);$u.Send(([byte[]](,0xFF*6)+($mac*16)),102)|Out-Null
💻 Linux / macOS (avancé, ligne de commande)

Installation :

  • Debian/Ubuntu : sudo apt install wakeonlan
  • macOS : brew install wakeonlan

Commande de réveil :

wakeonlan -i myserver.example.com -p 9 AA:BB:CC:DD:EE:FF
Toutes ces apps envoient exactement le même signal de réveil que le bouton de l'app principale. Si le réveil a déjà marché avant avec l'app, il marchera avec ces outils en utilisant les mêmes paramètres.
